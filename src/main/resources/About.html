Bootstrap App is a project created for Applied Java course.

This statistical tool allows you run different resampling schemes on previously generated observations to obtain bootstrap estimates of variance and 95% confidence intervals for both mean and median.

Read more about generating random observations

Generating random observations

Bootstrap App provides different data generating mechanisms. Independent observations can follow:

  • normal distribution,
  • exponential distribution.

Dependent observations can be generated as a stationary process:

  • moving average with standard normal noise.

In each case you can specify parameters of the chosen data generating mechanism and the number of observations.

Read more about bootstrap resampling schemes

Bootstrap resampling schemes

Bootstrap App implements four resampling schemes:

  • classic bootstrap (IID),
  • noneverlapping block bootstrap (NBB),
  • moving block bootstrap (MBB),
  • stationary bootstrap (SB).

For block bootstrap methods you can specify the block length. Since it is randomly chosen (it follows the geometric distribution) in the stationary bootstrap resampling scheme, user specifies the expected block length.
